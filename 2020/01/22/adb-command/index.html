例子

以qq为例

  1. 确保当前处于qq的窗口

  2. adb shell dumpsys window | grep -i focus

    得到qq包名

  3. adb shell pm path com.tencent.mobileqq

    得到apk路径

  4. adb pull /data/app/com.tencent.mobileqq-hzWsdlM5XmCPxSfMdY7Diw==/base.apk

    就得到qq的apk

Settings数据库中有三张表 system，global，secure

在Android 10上，Settings数据库以xml文件形式存储

路径为data/system/users/0

这个路径下有以下三个文件

settings_global.xml

settings_secure.xml

settings_system.xml

这些值就存在这些文件中

通过 get 表名【system ｜ global ｜ secure】 字段 获取值

通过put 表名【system ｜ global ｜ secure】 字段 设置值
